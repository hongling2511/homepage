## Self-Test Q&A

<details>
<summary>‚ùì What's the SDK's core design principle?</summary>

Give agents computer access via terminal, using the same tools humans use.

</details>

<details>
<summary>‚ùì Why is Bash the most powerful?</summary>

General + Composable + Store results + Dynamic scripts + Reuse existing software

</details>

<details>
<summary>‚ùì Two main benefits of subagents?</summary>

1. Parallelization
2. Isolated context, return only key info

</details>

<details>
<summary>‚ùì How to choose verification methods?</summary>

- Code ‚Üí Rules check (TypeScript/ESLint)
- UI ‚Üí Visual feedback (screenshots)
- Subjective ‚Üí LLM as judge (use carefully)

</details>
